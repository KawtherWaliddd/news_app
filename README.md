# News App 

A simple Flutter app that fetches and displays news articles from an API.

## Features
- Fetch news from NewsAPI
- Categories for filtering (e.g. Sports, Tech, Health)
- Responsive UI

## Tech Stack
- Flutter
- REST API (NewsAPI.org)
- Dio package

## Getting Started
1. Clone the repo
2. Run `flutter pub get`
3. Add your API key
4. Run the app
